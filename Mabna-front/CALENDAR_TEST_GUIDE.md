# راهنمای تست تقویم شمسی

## مراحل تست

### 1. ورود به سیستم
- وارد سایت شوید: http://localhost:5173
- با اکانت خود وارد شوید

### 2. دسترسی به تقویم
- در منوی کناری روی "تقویم" کلیک کنید
- تقویم شمسی باید نمایش داده شود

### 3. بررسی تاریخ فعلی
- تاریخ امروز باید مشخص باشد (با رنگ آبی)
- ماه و سال فعلی شمسی نمایش داده شود

### 4. تست ناوبری
- روی دکمه‌های قبل/بعد کلیک کنید
- ماه‌ها باید تغییر کنند
- روی "امروز" کلیک کنید تا به تاریخ فعلی برگردید

### 5. تست ایجاد رویداد
- روی "رویداد جدید" کلیک کنید
- فرم باید باز شود
- فیلدهای زیر را پر کنید:
  - عنوان: "تست رویداد"
  - توضیحات: "این یک تست است"
  - زمان شروع و پایان
  - دسته‌بندی و رنگ
- روی "ایجاد" کلیک کنید

### 6. تست نمایش رویداد
- رویداد باید در تقویم نمایش داده شود
- روی رویداد کلیک کنید
- جزئیات باید نمایش داده شود

### 7. تست ویرایش رویداد
- در مودال جزئیات، عنوان را تغییر دهید
- روی "ذخیره" کلیک کنید
- تغییرات باید اعمال شوند

### 8. تست حذف رویداد
- روی "حذف" کلیک کنید
- رویداد باید از تقویم حذف شود

## خطاهای احتمالی و راه‌حل

### خطای "Failed to resolve import"
```bash
cd frontend
npm install @radix-ui/react-label @radix-ui/react-select
npm run dev
```

### خطای moment-jalaali
```bash
npm install moment-jalaali
```

### خطای CSS
- مطمئن شوید که Tailwind CSS نصب است
- فایل index.css باید @tailwind directives داشته باشد

### خطای کامپوننت‌ها
- مطمئن شوید که تمام فایل‌های ui موجود هستند
- shadcn/ui کامپوننت‌ها باید درست import شوند
