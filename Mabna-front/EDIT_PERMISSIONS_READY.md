## عملکرد فعلی

### وظایف (Tasks):
```javascript
// اگر is_creator === true
- دکمه "ویرایش وظیفه" نمایش داده می‌شود ✅
- دکمه "حذف وظیفه" نمایش داده می‌شود ✅
```

```javascript
// اگر is_creator === true
```

## تست‌های پیشنهادی

### 1. تست سازنده وظیفه:
1. وارد صفحه وظایف شوید
2. روی وظیفه‌ای که خودتان ساخته‌اید کلیک کنید
3. باید دکمه‌های "ویرایش وظیفه" و "حذف وظیفه" را ببینید
4. روی "ویرایش وظیفه" کلیک کنید و تغییراتی اعمال کنید
5. تغییرات باید ذخیره شوند

### 2. تست غیر سازنده وظیفه:
1. روی وظیفه‌ای که شخص دیگری ساخته کلیک کنید  
2. نباید دکمه‌های ویرایش و حذف را ببینید
3. فقط امکان مشاهده و کامنت باید وجود داشته باشد

### 3. تست سازنده هدف:
1. وارد صفحه اهداف شوید
2. در کارت هدفی که خودتان ساخته‌اید باید دکمه‌های "ویرایش" و "حذف" را ببینید
3. باید بتوانید نتیجه کلیدی اضافه کنید

### 4. تست غیر سازنده هدف:
1. در کارت هدفی که شخص دیگری ساخته فقط دکمه "گزارش" باید باشد
2. نباید بتوانید نتیجه کلیدی اضافه کنید
