# پیاده‌سازی تقویم شمسی

## خلاصه

تقویم شمسی با استفاده از shadcn/ui و moment-jalaali پیاده‌سازی شده است.

## ویژگی‌های پیاده‌سازی شده

### ✅ تقویم شمسی
- نمایش تاریخ‌های شمسی با استفاده از moment-jalaali
- نمایش نام‌های فارسی ماه‌ها و روزهای هفته
- تاریخ امروز به‌صورت خودکار مشخص می‌شود
- ناوبری بین ماه‌ها با دکمه‌های قبل/بعد
- دکمه "امروز" برای بازگشت به تاریخ فعلی

### ✅ مدیریت رویدادها
- ایجاد رویداد جدید با مودال
- نمایش رویدادها در روزهای مربوطه
- ویرایش و حذف رویدادها
- دسته‌بندی رویدادها (جلسه، وظیفه، یادآوری، شخصی)
- رنگ‌بندی رویدادها (آبی، سبز، بنفش، نارنجی، صورتی، قرمز)

### ✅ رابط کاربری
- طراحی responsive برای موبایل و دسکتاپ
- استفاده از shadcn/ui components
- انیمیشن‌ها و hover effects
- راست‌چین (RTL) برای زبان فارسی

## وابستگی‌های مورد استفاده

### موجود در پروژه ✅
- `moment-jalaali`: برای کار با تاریخ شمسی
- `lucide-react`: برای آیکون‌ها
- `@radix-ui/*`: برای کامپوننت‌های پایه shadcn
- `class-variance-authority`: برای مدیریت کلاس‌ها
- `tailwind-merge`: برای ترکیب کلاس‌های Tailwind

### نصب شده در این پیاده‌سازی ✅
همه وابستگی‌های مورد نیاز از قبل در پروژه موجود بودند.

## نحوه استفاده

### 1. دسترسی از داشبورد
تقویم به منوی کناری داشبورد اضافه شده و با کلیک روی "تقویم" قابل دسترسی است.

### 2. ایجاد رویداد جدید
- روی دکمه "رویداد جدید" کلیک کنید
- فرم را پر کنید:
  - عنوان (اجباری)
  - توضیحات (اختیاری)
  - زمان شروع و پایان
  - دسته‌بندی
  - رنگ

### 3. مشاهده و ویرایش رویدادها
- روی هر رویداد در تقویم کلیک کنید
- مودال جزئیات باز می‌شود
- امکان ویرایش یا حذف رویداد

## ویژگی‌های تاریخ شمسی

### تاریخ فعلی
```javascript
// تاریخ امروز به‌صورت شمسی
const today = moment()
console.log(today.format('jYYYY/jMM/jDD')) // مثال: 1403/10/04
```

### نمایش ماه و سال
```javascript
// نمایش نام ماه و سال شمسی
const monthYear = currentDate.format('jMMMM jYYYY') // مثال: دی 1403
```

### تولید روزهای ماه
```javascript
// تولید تمام روزهای یک ماه شمسی
const startOfMonth = moment(currentDate).startOf('jMonth')
const endOfMonth = moment(currentDate).endOf('jMonth')
```

## اتصال به API

برای اتصال به بکند، کافی است توابع زیر را فعال کنید:

```javascript
// در Calendar.jsx
const fetchEvents = async () => {
  const response = await fetch(`${API_BASE_URL}/api/events`, {
    headers: { 'Authorization': `Bearer ${token}` }
  })
  // ...
}

const handleEventCreate = async (eventData) => {
  const response = await fetch(`${API_BASE_URL}/api/events`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(eventData)
  })
  // ...
}
```

## API Endpoints مورد نیاز

اگر بکند بخواهد پشتیبانی کند:

```
GET    /api/events           # دریافت لیست رویدادها
POST   /api/events           # ایجاد رویداد جدید
PUT    /api/events/:id       # ویرایش رویداد
DELETE /api/events/:id       # حذف رویداد
```
