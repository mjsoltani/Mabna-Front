# تست قابلیت ویرایش وظایف و اهداف

## سناریوهای تست

### 1. تست وظایف (Tasks)

#### سناریو A: کاربر سازنده وظیفه
```
شرایط: is_creator: true
انتظار:
- دکمه "ویرایش وظیفه" نمایش داده شود
- دکمه "حذف وظیفه" نمایش داده شود  
- دکمه "✏️ ویرایش توضیحات" نمایش داده شود
- مودال ویرایش باز شود
- تمام فیلدها قابل ویرایش باشند
```

#### سناریو B: کاربر غیر سازنده وظیفه
```
شرایط: is_creator: false یا undefined
انتظار:
- دکمه‌های ویرایش و حذف مخفی باشند
- فقط امکان مشاهده و کامنت وجود داشته باشد
- تغییر وضعیت همچنان امکان‌پذیر باشد (برای assignee)
```

### 2. تست اهداف (Objectives)

#### سناریو A: کاربر سازنده هدف
```
شرایط: is_creator: true
انتظار:
- دکمه "ویرایش" نمایش داده شود
- دکمه "حذف" نمایش داده شود
- دکمه "+ نتیجه کلیدی" نمایش داده شود
- دکمه‌های ویرایش/حذف نتایج کلیدی نمایش داده شوند
```

#### سناریو B: کاربر غیر سازنده هدف
```
شرایط: is_creator: false یا undefined
انتظار:
- فقط دکمه "گزارش" نمایش داده شود
- دکمه‌های ویرایش، حذف و اضافه کردن نتیجه کلیدی مخفی باشند
- فقط دکمه "📊" (گزارش) برای نتایج کلیدی نمایش داده شود
```

## چک‌لیست تست

### وظایف:
- [ ] مودال ویرایش وظیفه باز می‌شود
- [ ] تمام فیلدها با مقادیر فعلی پر می‌شوند
- [ ] تغییرات ذخیره می‌شوند
- [ ] حذف وظیفه کار می‌کند
- [ ] تایید حذف نمایش داده می‌شود
- [ ] ویرایش توضیحات inline کار می‌کند
- [ ] دکمه‌ها برای غیر سازندگان مخفی هستند

### اهداف:
- [ ] مودال ویرایش هدف باز می‌شود
- [ ] تاریخ‌ها درست نمایش داده می‌شوند
- [ ] حذف هدف کار می‌کند
- [ ] اضافه کردن نتیجه کلیدی فقط برای سازنده
- [ ] ویرایش نتایج کلیدی فقط برای سازنده
- [ ] حذف نتایج کلیدی فقط برای سازنده

## نکات مهم برای تست

1. **بررسی API Response**: مطمئن شوید که `is_creator` در response موجود است
2. **تست بدون is_creator**: اگر فیلد موجود نباشد، دکمه‌ها نباید نمایش داده شوند
3. **تست Responsive**: در موبایل دکمه‌ها درست نمایش داده شوند
4. **تست Error Handling**: اگر ویرایش ناموفق باشد، خطای مناسب نمایش داده شود

## مثال داده‌های تست

### وظیفه با سازنده:
```json
{
  "id": "task-1",
  "title": "وظیفه تست",
  "is_creator": true,
  "assignee": {...},
  "status": "todo"
}
```

### وظیفه بدون سازنده:
```json
{
  "id": "task-2", 
  "title": "وظیفه دیگران",
  "is_creator": false,
  "assignee": {...},
  "status": "in_progress"
}
```

### هدف با سازنده:
```json
{
  "id": "obj-1",
  "title": "هدف تست",
  "is_creator": true,
  "key_results": [...]
}
```
